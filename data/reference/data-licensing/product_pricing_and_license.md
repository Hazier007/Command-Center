# Data Licensing Bundle (BE) — product outline + pricing + license

## Product page outline (1-pager)

### 1) Hero
- **Titel:** Belgium Business Rules Data Pack (VAT + Holidays + Mileage)
- **Subtitel:** Versioned JSON datasets with effective dates + changelogs — ready for calculators, payroll tooling, and SaaS apps.
- **CTA:** Download sample JSON + Request license / subscribe

### 2) What you get (datasets)
- `be_tax_rules.json`
  - VAT rate tiers (6/12/21)
  - Curated notable changes with **effectiveFrom** dates
  - Changelog for updates
- `be_workdays_holidays.json`
  - Statutory public holidays (per year)
  - Workweek defaults + replacement-holiday rule stub
- `be_km_allowance.json`
  - Mileage allowance maxima (annual + quarterly systems)
  - Effective dates + notes

### 3) Who it’s for
- Tool sites (calculators)
- Payroll / HR software
- Accounting apps
- Agencies building internal ops tools

### 4) Guarantees / update cadence
- **Versioned releases** (schemaVersion)
- **Effective dates** on every rate entry
- Updates whenever Belgium publishes new indexations / changes (best-effort SLA)
