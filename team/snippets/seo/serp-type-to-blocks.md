# SERP type → page blocks (quick mapping)

Wanneer gebruiken: bij programmatic/local pages om structuur snel te kiezen.

## local_pack_dominant
- Hero + CTA boven de fold (call/form)
- Trust chips (snelheid/transparant/lokaal)
- Services bullets (kort)
- Werkwijze (3 stappen)
- Local coverage ({city}+omgeving)
- FAQ (8–10) + FAQPage schema
- Disclaimer (matchen met partner)

## organic_dominant
- Intro (problem framing)
- Sections met H2/H3 (meer uitleg)
- FAQ uitgebreider
- Related tools/content links
- Schema afhankelijk van type (FAQPage/Article)

Input: keyword + SERP quickscan.
Output: gekozen blocks + spec in handoff.
