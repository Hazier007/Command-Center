# Team Snippets Library

Doel: herbruikbare stukjes die ons elke week uren sparen.

Regels:
- Voeg enkel iets toe als het **minstens 2× hergebruikt** is.
- Elke snippet eindigt met: **wanneer gebruiken** + **input** + **output**.

Structuur:
- `seo/` — keyword/intent/links/spec
- `dev/` — Next.js/programmatic/deploy
- `ops/` — defaults/pricing/error contracts/release
