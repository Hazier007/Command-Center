# Definition of Done

## Ship-ready means
Bart (Obie) should only need to review whether the output:
- looks good
- is aligned with the project
- is acceptable to ship

## Done does NOT mean
- planned
- drafted
- partially built
- explained but not executed
- claimed without proof

## Done means
- working
- tested
- structurally correct
- useful in practice
- reviewed by Lisa

## Team interpretation
- **Wout:** research is done when findings are real, relevant, and decision-ready
- **JC:** implementation is done when it works and has been tested
- **Lisa:** nothing is done until it is usable for Obie
