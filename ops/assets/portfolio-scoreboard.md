## Priority framing

### Tier 1 - highest leverage now
- assets with traffic/revenue history
- assets that can move fastest toward recurring income
- assets that already exist and only need sharper execution

### Tier 2 - strong opportunity but needs validation/build
- planned assets with credible monetization angle
- calculators / directories / leadgen sites with clear CPC or commercial intent

### Tier 3 - speculative / later
- long-tail domain ideas without immediate path to shipping
- dormant or unclear assets that need re-evaluation before work starts

## Next operational move
Create and maintain one canonical inventory where every domain/site gets:
- type
- status
- monetization model
- project linkage
- owner
- next action
- priority tier
