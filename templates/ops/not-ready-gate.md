# NOT READY Gate (startcriteria)

Een taak mag pas starten als de handoff dit bevat:
- Repo + branch + push policy (PR/direct)
- Deploy target
- Constraints (taal/regio/defaults)
- Input schema (CSV/JSON velden)
- DoD/acceptance criteria
- NIET in scope (max 3 bullets)

Rule: geen gate = geen start.
