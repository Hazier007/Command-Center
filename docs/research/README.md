# Research vault

Alle keyword research / SEO research wordt hier opgeslagen zodat het team het altijd kan terugvinden.

## Naamgeving
`YYYY-MM-DD__project__onderwerp.md`

Voorbeeld:
- `2026-02-11__opblaasbareboot__keywords+template.md`

## DataForSEO
We gebruiken (optioneel) DataForSEO **Keywords Data** (Google Ads) om **zoekvolume/CPC/competition** op te halen per keyword.

Verwachte env vars:
- `DATAFORSEO_LOGIN`
- `DATAFORSEO_PASSWORD`

> Let op: DataForSEO werkt met **API login + API password** (basic auth). Dit is niet altijd hetzelfde als “API key”.
