## 🛠️ Tech Stack

### **Leadgen Sites**
- **CMS:** WordPress (headless) or Next.js
- **Hosting:** Hostinger/Cloudways (€20-50/site/month)
- **SEO:** Programmatic pages (city × service templates)
- **Forms:** Typeform or custom (webhook to CRM)
- **CRM:** Airtable (free) → upgrade to HubSpot/Pipedrive later

### **Intel Platform**
- **Data sources:** DataForSEO API, custom scrapers
- **Database:** PostgreSQL (store historical data)
- **Reports:** Markdown templates → PDF (Puppeteer)
- **Delivery:** Email automation (SendGrid/Mailchimp)
- **Dashboard:** Next.js + Charts.js (optional Phase 2)

### **Automation**
- **Cron jobs:** Data collection (daily/weekly)
- **OpenClaw skills:** DataForSEO, web scraping, report generation
- **Monitoring:** Uptime checks, ranking trackers, anomaly alerts

## ⚠️ Risks & Mitigation

### **Risk 1: Google Algorithm Changes**
**Impact:** Rankings drop, traffic loss  
**Mitigation:**
- Diversify across 10+ niches (not reliant on 1 site)
- Quality content + user intent focus (Google-proof)
- Backlink strategy (authority signals)

### **Risk 2: Lead Buyer Churn**
**Impact:** Lose revenue if buyers stop purchasing  
**Mitigation:**
- 3-5 buyers per niche (redundancy)
- Quality leads (screen before selling)
- Contracts with minimums (€500/month commitment)

### **Risk 3: Tech Failures**
**Impact:** Sites down, data loss  
**Mitigation:**
- Automated backups (daily)
- Uptime monitoring (alerts)
- Jean-Cloud on-call (SLA: <4h fix time)

### **Risk 4: Copycat Competitors**
**Impact:** Someone clones our model  
**Mitigation:**
- Data moat (historical trends, hard to replicate)
- Speed (launch 10+ sites before competitors react)
- Network effects (lead buyers locked in)
